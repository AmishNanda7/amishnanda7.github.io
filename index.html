⭐ Featured · Data Analysis + Machine Learning
LA House Price & Mortgage Rate Predictor
⌥

An end-to-end data analysis and machine learning project predicting next month's Los Angeles home values using real-world Zillow ZHVI housing data and FRED 30-year mortgage rate data. Built a full pipeline from raw data to an interactive Streamlit dashboard with live prediction inputs.

  • Merged two real-world datasets (Zillow Metro ZHVI + FRED MORTGAGE30US) with date alignment, reshaping, and monthly aggregation
  • Engineered lag features (ZHVI_Lag1, Rate_Lag1) to capture temporal dependencies for next-month prediction
  • Compared Linear Regression vs Random Forest — Linear Regression achieved R² = 0.9982, MAE = $2,365, RMSE = $2,856, significantly outperforming Random Forest for this time-series problem
  • Built a full Streamlit dashboard with live prediction sidebar, 4 interactive charts (trend, mortgage rates, scatter, actual vs predicted), model metrics, and CSV data download
  • Presented findings with a recorded video presentation covering methodology, model selection rationale, and results
